# Keyboard layouts

I base my 2 layouts on Portuguese PC layout for both Windows and MacOS.

## PT+

PT (PC) layout to be able to write portguese with a couple changes:

- `Alt Gr` + `+` -> `[`
- `Alt Gr` + `´` -> `]`
- `≪` / `≫` -> `+` / `*`
- `º` / `ª` -> `` ` `` / `@`

## PT-Dev

Everything that PT+ does, removes all dead keys, and:

- `+` -> `(`
- `´` -> `)`
- `Shift` + `+` -> `}`
- `Shift` + `´` -> `}`
- `ç` -> - `;`
- `Shift` + `ç` -> - `:`
